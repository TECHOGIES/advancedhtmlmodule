HTML 5 Introduction

What is HTML5?

HTML5, short for HyperText Markup Language version 5, is the latest standard used to create and design web pages. It's easy to learn and offers a lot of new features for modern web development.

HTML describes the structure of a webpage, defining elements like headers, paragraphs, images, links, and more.

HTML5 is the fifth and most recent major version of HTML, bringing enhancements and features that improve the user experience and developer productivity.

Why Learn HTML5?

If you're serious about web development, learning HTML is non-negotiable. It's the foundation of any website and a prerequisite to learning other key web technologies like CSS and JavaScript.

Example:

Below is a simple HTML code that shows a "Hello World" message:

<!DOCTYPE html>
<html>
<head>
   <title>HELLO WORLD!</title>
</head>
<body>
   <h1>Lorem ipsum dolor sit amet</h1>
   <p>This is a paragraph</p>
</body>
</html>

Example Explained

<!DOCTYPE html>: Declares that the document is an HTML5 document.

<html>: The root element that encloses all HTML elements on the page.

<head>: Contains meta-information about the document, such as styles and scripts.

<title>: Sets the title of the webpage, which appears in the browser tab.

<body>: Contains the visible content of the page, such as text, images, and links.

<h1>: Defines the largest heading level (Heading 1).

<p>: Defines a paragraph of text.

HTML Tags

HTML tags are the building blocks of web pages, surrounded by angle brackets. Each tag usually comes in pairs, an opening tag and a closing tag, as shown below:

<p>Hello, Let's Learn HTML!</p>

Start Tag and End Tag

  • Start Tag: Also called an opening tag, like <p>.
  • End Tag: Also called a closing tag, like </p>.

Basic Structure of an HTML Page:

</head>
<body>
</body>
